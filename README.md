# Configurações WhatsApp Business - Pousada Praia de Ponta Negra

Bem-vindo ao repositório! Aqui estão todas as configurações para o WhatsApp Business da pousada. Use para backup, edição ou compartilhamento com a equipe.

## Como Usar
1. Crie ou edite os arquivos conforme necessário.
2. No WhatsApp Business (app no celular):
   - Abra Configurações.
   - Adicione respostas rápidas de `respostas-rapidas.md`.
   - Configure mensagens automáticas de `mensagens-automaticas.md`.
3. Teste: Peça a alguém para mandar mensagem e veja se funciona.
